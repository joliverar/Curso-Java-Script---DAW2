Guía de estilos 1. Tipografía Se utiliza la tipografía por defecto de Bootstrap (system-ui). Los títulos usan mayor tamaño y peso. Los párrafos usan color text-muted para lectura cómoda. 2. Colores Fondo hero y navbar: gris (bg-secondary) Fondo alternativo: bg-light Texto principal: negro Botones principales: azul Bootstrap (btn-primary) Bordes y separación: clases Bootstrap (border) 3. Estructura Responsive Se utiliza el sistema grid de Bootstrap: col-12 en móviles (XS) col-md-6 en tablet (MD) col-xl-6 en escritorio (XL) En móvil los elementos se apilan verticalmente. En escritorio se muestran en dos columnas. 4. Interactividad Slider automático con 3 imágenes sin controles. Imagen Smart Waste cambia de color a blanco y negro mediante hover usando transición CSS. 5. Usabilidad Formularios con etiquetas asociadas. Todas las imágenes incluyen atributo alt. Navegación clara y estructura semántica.
